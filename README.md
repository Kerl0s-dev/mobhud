# MobHUD

MobHUD is a lightweight client-side mod that displays information about the entity you are currently looking at.

When aiming at a mob or any living entity, a clean HUD appears at the top of the screen showing the entity's **name** and **health bar**.

## Features

* Displays the **entity name**
* Shows a **health bar**
* Appears only when **looking at an entity**
* Clean and **minimal design**
* Lightweight and **client-side only**

## Installation

1. Install **Fabric Loader**
2. Install **Fabric API**
3. Download the MobHUD `.jar`
4. Place it in your `mods` folder

## Compatibility

* Works with most **living entities**
* Client-side only
* Built using **Fabric**
